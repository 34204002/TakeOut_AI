# SkyTakeOut 外卖订单全链路管理系统
基于Spring Boot开发的电商外卖全流程业务平台，覆盖C端用户下单、B端商家接单、管理端运营全场景，完整实现菜品管理、订单生命周期管理、支付流程闭环、数据统计报表等核心功能，保障电商订单场景下的数据一致性与系统高可用。

## 技术栈
后端：Spring Boot、MyBatis、MySQL、Redis、RabbitMQ、WebSocket、POI
核心方向：订单全链路管理、缓存优化、消息异步解耦、数据统计、分布式事务保障

## 核心功能&个人二次开发
1.  订单全链路闭环管理
    - 实现下单、接单、配送、完成/取消全流程状态流转
    - 基于RabbitMQ延迟队列+死信队列，实现订单超时未支付自动取消+库存回滚
    - 基于可靠消息最终一致性方案，保障订单-库存数据一致性，杜绝资损风险
2.  缓存与性能全链路优化
    - 基于Redis+Spring Cache实现菜品、套餐数据多级缓存策略，解决高频访问数据库压力
    - 菜品查询接口响应时间优化60%，QPS从250提升至1100
3.  数据运营与实时通知能力
    - 基于POI实现多维度Excel报表导出，支持营业额、订单、菜品销量等核心指标统计
    - 基于WebSocket实现订单状态实时推送，商家端新订单通知延迟低于100ms
4.  支付闭环与稳定性建设
    - 对接微信支付沙箱环境，实现支付回调、退款全流程闭环
    - 基于定时任务+分布式锁实现订单状态定时处理，避免集群环境重复执行
    - 完善接口幂等性设计，杜绝重复下单问题，全局异常处理与日志追踪体系
